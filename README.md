## 介绍
该组件内置了更新版本的 Qemu 并不与系统源内的 Qemu 冲突，让 deepin/UOS 吃上更新版本的 Qemu  
（内置了支持龙芯新世界的 qemu-system-loongarch64）  
安装此包后在后续版本的 Wine 运行器虚拟机配置工具将支持调用该新版本的 Qemu（当前的 Wine 运行器 3.9.1 暂无法调用）
